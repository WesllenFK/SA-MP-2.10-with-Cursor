# Documentação Temporária

Esta pasta contém documentações de análise e arquitetura do projeto.

## Branch de Documentação

**Branch:** `docs/documentation`

Toda documentação deve ser commitada nesta branch específica.

## Arquivos

| Arquivo | Descrição |
|---------|-----------|
| `ARCHITECTURE_PLAN.md` | Plano de arquitetura geral do projeto |
| `MIGRATION_ANALYSIS.md` | Análise de migração de código |
| `SUBSYSTEMS_ARCHITECTURE.md` | Arquitetura dos subsistemas |
| `WEAPON_MODEL_LOADING_ANALYSIS.md` | Análise do sistema de carregamento de armas |
| `WEAPON_SYSTEM_DETAILED_TREE.md` | Árvore detalhada do sistema de armas |

## Como Contribuir

1. Fazer checkout da branch `docs/documentation`
2. Adicionar/editar arquivos `.md` na pasta `temporarios/`
3. Commit e push para a branch

```bash
git checkout docs/documentation
# editar arquivos
git add temporarios/
git commit -m "docs: descrição"
git push
```
